# ฟีเจอร์: ส่งรูปรายงานเช็คชื่อเข้า LINE

อัปเดต: 10 มิ.ย. 2569 (2026) — ไฟล์ที่แก้: `index-appscript.html`

## สรุปสั้น

ปุ่ม **🖼️ ส่งรูปเข้า LINE** ในแท็บเช็คชื่อ จะแปลงรายงานสวยๆ (หน้าตาเดียวกับ Export) ให้เป็น **รูปภาพ PNG** แล้วส่ง/บันทึกเพื่อแนบเข้า LINE ได้ทันที — ไม่ต้องตั้งค่า backend ใดๆ

## วิธีใช้

1. เลือกวันที่ที่มีข้อมูล
2. กดปุ่ม 🖼️ ส่งรูปเข้า LINE
3. ผลลัพธ์ขึ้นกับอุปกรณ์:
   - **มือถือ** → เด้งแผ่นแชร์ของเครื่อง เลือก LINE → เลือกกลุ่มผู้ปกครอง → ส่ง
   - **คอมพิวเตอร์** → บันทึกรูป PNG ลงเครื่อง แล้วเปิด LINE ลากรูปแนบเอง

## หน้าตารูป (รายงาน 1 หน้าเต็ม)

- หัวรายงานสีเลือดหมู + ชื่อโรงเรียน + วันที่ (แปลงเป็นไทย/พ.ศ. อัตโนมัติ)
- กล่องสรุป 4 สถานะ: ✅ มาเรียน / ⏰ มาสาย / 📝 ลา / ❌ ขาด
- แถบ % อัตราการเข้าเรียน (เขียว ≥80% / เหลือง ≥60% / แดง <60%)
- ตารางรายชื่อครบ 36 คน พร้อมป้ายสีสถานะ
- ท้ายรายงาน: ลายเซ็นระบบ + เตือนถ้ามีคนยังไม่บันทึกสถานะ

## หลักการทำงาน (เชิงเทคนิค)

1. `getReportContext()` ตรวจวันที่ + ว่ามีข้อมูลไหม
2. `buildReportHtml()` สร้าง HTML รายงาน (ใช้ร่วมกับปุ่ม Export)
3. เรนเดอร์ใน iframe ซ่อน (กัน CSS ชนหน้าเว็บหลัก)
4. โหลด `html2canvas` แบบ lazy จาก cdnjs (ครั้งแรกต้องต่อเน็ต)
5. แปลงเป็น PNG (scale 2 เท่า ภาพคมชัด)
6. ส่งผ่าน `navigator.share({files})` (มือถือ) หรือ fallback ดาวน์โหลด (คอม)

## ข้อควรรู้ / ข้อจำกัด

- ครั้งแรกที่กดต้องต่ออินเทอร์เน็ต (โหลด html2canvas)
- ปุ่ม 📄 Export รายงาน (ดาวน์โหลด .html) ยังอยู่ เผื่อพิมพ์/เก็บบนคอม
- ทุกครั้งที่แก้โค้ดต้อง Deploy เวอร์ชันใหม่: Deploy → Manage deployments → ✏️ → New version → Deploy (URL คงเดิม ห้ามกด New deployment)

## ประวัติทางเลือกที่เคยพิจารณา

- LINE Notify — ปิดบริการแล้ว (เม.ย. 2025) ใช้ไม่ได้
- แชร์ข้อความสรุป (`line.me/R/share?text=`) — เคยทำ แต่ส่งได้แค่ข้อความ ไม่สวย
- LINE Messaging API push อัตโนมัติ — ทำได้แต่ต้องตั้งค่า token/group ยุ่งยาก
- **เลือก: แปลงรายงานเป็นรูปแล้วแชร์** — สวยเท่า Export + ไม่ต้องตั้งค่า
