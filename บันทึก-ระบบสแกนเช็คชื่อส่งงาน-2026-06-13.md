# บันทึก: ระบบสแกนเช็คชื่อและส่งงานด้วยกล้องมือถือ (QR / Barcode Scanner)

**วันที่:** 13 มิถุนายน 2569 (2026)

## สิ่งที่ทำ

1. เพิ่มปุ่มสแกนเนอร์สำหรับเช็คชื่อนักเรียนและตรวจงาน/ข้อสอบ ผ่านกล้องมือถือหรือกล้องเว็บแคม โดยใช้รหัสนักเรียน (จากทะเบียน) หรือเลขที่บนตัวการ์ดเพื่ออัปเดตข้อมูลแบบเรียลไทม์ พร้อมระบบเสียงแจ้งเตือนออฟไลน์
2. **ระบบจัดทำการ์ด QR Code ประจำตัวนักเรียน (`qr-card.html`)**: สร้างระบบสร้างการ์ดสําหรับพิมพ์แจกนักเรียน เพื่อใช้สแกนทำกิจกรรมต่างๆ
3. **เครื่องมือสร้าง QR Code ทั่วไป (`qr-generator.html`)**: พัฒนาเครื่องมือสร้าง QR Code สไตล์พรีเมียมแบบครบวงจร รองรับ ลิงก์, ข้อความ, Wi-Fi, เบอร์โทร, SMS ปรับเปลี่ยนลายจุด สีเฉดโลโก้ และคัดลอก/ดาวน์โหลดทันที
4. **เชื่อมโยงจุดนำทางแบบฝังตัว**: อัปเดตเมนูด้านข้าง (Drawer Navigation) ของทุกเว็บไซต์ และแท็บจัดการนักเรียนของหน้าส่งงาน เพื่อเชื่อมไปยังหน้าสร้างการ์ด ทั้ง 2 รูปแบบ ป้องกันลิงก์เว็บสูญหาย
5. **แก้ปัญหาลิงก์ภาษาไทยด้วยการใช้ชื่อไฟล์ภาษาอังกฤษ**: เปลี่ยนชื่อไฟล์ระบบทำ QR ทั้งหมดเป็นภาษาอังกฤษ เพื่อป้องกันการเกิด error 404 File Not Found ตอนนำขึ้นเซิร์ฟเวอร์ออนไลน์

## รายละเอียดทางเทคนิค

1. **ไลบรารีสแกนเนอร์**: โหลดสคริปต์ `html5-qrcode` ผ่าน CDN ในส่วน `<head>` ของทุกไฟล์
2. **ระบบเสียงตอบรับออฟไลน์**: สังเคราะห์เสียงผ่าน **Web Audio API** บนบราวเซอร์ ไม่ต้องดาวน์โหลดไฟล์ MP3 ทำงานได้แม้ไม่มีอินเทอร์เน็ต
   - สแกนสำเร็จ: เสียงติ๊ดสูง (950Hz)
   - สแกนผิดพลาด / ไม่พบข้อมูล: เสียงอือต่ำ (180Hz)
3. **ตรรกะการจับคู่รหัส (Matching Logic)**:
   - ตรวจสอบรหัสนักเรียน 4-5 หลักจากสแกนเนอร์ก่อน หากตรงกับในระบบจะอัปเดตให้ทันที
   - หากรหัสไม่ตรง แต่เป็นตัวเลขธรรมดา (เช่น `"5"`) ระบบจะจับคู่เลขที่นั้นในตารางให้อัตโนมัติ (ช่วยให้คุณครูสามารถพิมพ์บาร์โค้ดเลขที่ธรรมดาให้นักเรียนพกได้)
4. **กลไกการสแกนบนมือถือ**:
   - เลือกกล้องหลัง (Back/Rear Camera) เป็นค่าเริ่มต้นโดยอัตโนมัติ และมีดรอปดาวน์ให้สลับกล้องได้
   - ตั้งหน่วงเวลาสแกนซ้ำ (Scan Cooldown) 1.8 วินาที เพื่อไม่ให้ระบบอ่านค่าการ์ดเดิมซ้ำซ้อนขณะส่องกล้อง
   - ในระบบตรวจงาน หากเป็นงานประเภท "ข้อสอบ" จะมีช่องให้พิมพ์คะแนนเพื่อบันทึกคะแนนสอบลงระบบทันที
5. **ระบบสร้างการ์ดประจำตัว (`qr-card.html`)**:
   - ดึงข้อมูลจากไฟล์ `roster.json` ซึ่งเป็นไฟล์ข้อมูลนักเรียนประจำปี 2569
   - มีเมนูให้ครูเลือกธีมสีประจำห้องเรียน และเลือกว่าจะใช้ค่าใน QR Code เป็นรหัสนักเรียนหรือเลขที่ธรรมดา
