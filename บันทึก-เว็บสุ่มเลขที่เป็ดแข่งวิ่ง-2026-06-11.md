# บันทึก: เว็บสุ่มเลขที่ เป็ดแข่งวิ่ง 🦆 (11 มิ.ย. 2569)

## ข้อมูลหลัก

| รายการ | รายละเอียด |
|---|---|
| เว็บจริง | **https://kunewnew.github.io/duckrace/** |
| repo | `kunewnew/duckrace` (GitHub Pages จาก branch main) |

## วิธีใช้

1. เลือกห้อง → กด **🏁 ปล่อยเป็ด!** → นับ 3-2-1
2. เป็ดวิ่งเข้าเส้นชัย ตัวแรกที่เข้า = เลขที่ที่ถูกสุ่ม ขึ้นการ์ดเลขที่+ชื่อ+คอนเฟตตี
3. การสุ่มยุติธรรมเสมอ — ระบบสุ่มผู้ชนะแบบ uniform ก่อน แล้วค่อยให้เป็ดตัวนั้นเข้าเส้นชัยก่อน

## ฟีเจอร์

- **เป็ดหน้าตาสุ่ม**: ตา 6 แบบ / หมวก 7 แบบ / สีตัว 9 สี + มุกระหว่างวิ่ง (หมุนตัว กระโดด หันหลัง ร้องก๊าบ) ป้ายเลขที่ติดตัวเป็ด มงกุฎ 👑 ตามผู้นำ
- **เวอร์ชันมือถือ**: สนามยาวกว่าจอ ~3 เท่า กล้องวิ่งตามกลุ่มนำ + minimap แถบบนบอกตำแหน่ง/เลขผู้นำ + slow-motion ซูมโค้งสุดท้าย + แฟลช/จอสั่นตอนเข้าเส้น + แถบพากย์ประกาศการแซง
- **เสียงลุ้น (WebAudio สังเคราะห์ ไม่ใช้ไฟล์เสียง)**: ฝีเท้ารัวเร่งตามระยะ เสียงเชียร์ดังขึ้น โทนไต่ระดับ หัวใจเต้นตึกๆ ใกล้เส้นชัย เฮตอนแซง แฟนแฟร์ตอนเข้าเส้น — ปุ่ม 🔊/🔇 จำค่าไว้
- **โหมดไม่สุ่มซ้ำ**: สวิตช์เปิด-ปิด (ค่าเริ่มต้น = เปิด) จำประวัติต่อห้องใน localStorage, chips โชว์เลขที่สุ่มแล้ว, ปุ่ม ↺ รีเซ็ต, สุ่มครบทุกคน → ชวนรีเซ็ตรอบใหม่
- **เมนู ☰** เชื่อม 3 เว็บ: เป็ดแข่งวิ่ง / เช็คชื่อรายวิชา / เช็คชื่อเตรียมวิศวะ

## localStorage keys

- `duckrace_norepeat` — เปิด/ปิดโหมดไม่สุ่มซ้ำ ('1'/'0')
- `duckrace_used_{ห้อง}` — เลขที่ที่สุ่มไปแล้วของแต่ละห้อง เช่น `duckrace_used_6_1`
- `duckrace_mute` — ปิดเสียง ('1' = ปิด)

## การ deploy วันนี้

- สร้าง repo `duckrace` + อัปโหลด index.html + เปิด Pages (ผ่าน Claude in Chrome)
- เพิ่มลิงก์เป็ดแข่งวิ่งเข้าเมนู ☰ ของอีก 2 เว็บ:
  - repo `checkinSD` — commit `e93e9d6` ✅
  - repo `checkin` — commit `3bca016` ✅
  - ไฟล์ local อัปเดตครบ 3 ไฟล์ (เช็คชื่อรายวิชา, github-pages, index-appscript)

## ⚠️ งานค้าง

**เว็บเตรียมวิศวะเวอร์ชัน Apps Script** — ไฟล์ `เว็บ/index-appscript.html` มีลิงก์เป็ดแข่งวิ่งแล้ว แต่ต้องเอาไปวางทับใน Apps Script editor แล้วกด **Deploy → New version** เอง เมนูฝั่งนั้นถึงจะโชว์ลิงก์ใหม่

## ถ้าจะแก้ในอนาคต

- รายชื่อนักเรียนเปลี่ยน → regenerate ROOMS_DATA จาก `เช็คชื่อรายวิชา/index.html` แล้วแทนที่ในไฟล์นี้
- logic หลักอยู่ใน script ท้ายไฟล์: `setupTrack()` (สร้างสนาม/เป็ด), `startRace()` (วิ่ง+กล้อง+เสียง), `finishRace()` (บันทึกผล)
- แก้เสร็จอัปไฟล์ใหม่ทับ index.html ใน repo `duckrace` (Pages redeploy เอง ~1 นาที)
