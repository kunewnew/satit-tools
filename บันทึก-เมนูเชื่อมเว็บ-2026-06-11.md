# บันทึก: เมนูซ่อนฝั่งซ้าย (☰) เชื่อมเว็บเช็คชื่อทั้งสองเว็บ

**วันที่:** 11 มิถุนายน 2569

## สิ่งที่ทำ

เพิ่มปุ่ม **☰** มุมซ้ายบนทั้งสองเว็บ กดแล้วแถบเมนูเลื่อนออกจากฝั่งซ้าย แสดงรายการเว็บทั้งหมด กดพื้นหลังมืดหรือ ✕ เพื่อปิด (เว็บปัจจุบันแสดงจางๆ กดไม่ได้)

| เว็บ | ไฟล์ที่แก้ | ตำแหน่งปุ่ม ☰ |
|---|---|---|
| เช็คชื่อรายวิชา (ครูทุกคน) | `เช็คชื่อรายวิชา/index.html` | header ซ้ายสุด |
| เช็คชื่อเตรียมวิศวะ | `github-pages/index.html` + `index-appscript.html` | มุมซ้ายบนหน้าเลือกห้อง + header |

## ลิงก์เว็บ (ใช้งานจริง)

- เช็คชื่อเตรียมวิศวะ: https://kunewnew.github.io/checkin/ (repo `checkin`)
- เช็คชื่อรายวิชา: https://kunewnew.github.io/checkinSD/ (repo `checkinSD`)

## วิธีเพิ่มเว็บใหม่เข้าเมนู

แก้ array `SITE_LINKS` ท้ายไฟล์ (ก่อน `</script>`) **ทุกไฟล์ทั้ง 3 ไฟล์** เพิ่ม 1 บรรทัด:

```js
{icon:'🏫', label:'ชื่อเว็บใหม่', url:'https://...'},
```

- `url:''` (ว่าง) = เว็บปัจจุบัน จะแสดงจางๆ กดไม่ได้
- แก้เสร็จต้องอัปไฟล์ขึ้น GitHub ใหม่ (และ Apps Script ถ้าใช้)

## ปัญหาที่เจอ + วิธีแก้

**อาการ:** กดเมนูจากเว็บรายวิชา → ไปเว็บเตรียมวิศวะได้ แต่กดจากเว็บเตรียมวิศวะ → ไม่ไปไหน (404)

**สาเหตุ:** เมนูชี้ไป `kunewnew.github.io/checkin-subjects/` แต่ repo นั้นอัปไฟล์ผิด (เป็น `index-appscript.html` ของเว็บเตรียมวิศวะ ไม่ใช่ `index.html` ของเว็บรายวิชา) ส่วนเว็บรายวิชาตัวจริงอยู่ที่ repo `checkinSD`

**วิธีแก้:** เปลี่ยน URL ใน `SITE_LINKS` ของเว็บเตรียมวิศวะเป็น `https://kunewnew.github.io/checkinSD/`

## ขั้นตอนที่ต้องทำหลังแก้ (checklist)

- [ ] อัป `github-pages/index.html` ทับใน repo `checkin` (Add file → Upload files → Commit → รอ 1–2 นาที)
- [ ] ลบ repo `checkin-subjects` ทิ้ง (Settings → Delete this repository) — กันสับสน
- [ ] (ถ้าใช้เว็บผ่าน Apps Script) paste `index-appscript.html` ทับไฟล์ index → Deploy → Manage deployments → ✏️ → New version

## หมายเหตุเทคนิค

- โครงสร้าง drawer: `#drawer` + `#drawer-backdrop` + ฟังก์ชัน `openDrawer()/closeDrawer()/renderDrawerLinks()`
- ดีไซน์คุมโทนตามแต่ละเว็บ: รายวิชา = เอิร์ธโทนครีม/น้ำตาล, เตรียมวิศวะ = เลือดหมูเข้ม
- ตรวจ JS syntax ผ่านทั้ง 3 ไฟล์ (node --check)
