## ฟีเจอร์หลัก
1. **สร้างห้อง/วิชา** เอง — เลือกห้องจากทะเบียน หรือพิมพ์ใหม่ สลับห้องด้วยปุ่มชิป
2. **งาน 2 ชนิด**
   - งานส่ง → ช่องติ๊ก ส่ง (เขียว) / ไม่ส่ง (แดง) คลิกสลับ
   - ข้อสอบ → ช่องกรอกคะแนน (ตั้งคะแนนเต็มได้)
3. **ตารางงาน** นักเรียน × งาน + คอลัมน์นับ "ค้างส่ง" อัตโนมัติ
4. **แถวสรุปท้ายตาราง (📊 สรุปรายงาน)** — ติดขอบล่างเวลาเลื่อน บอกต่อคอลัมน์งาน:
   งานส่ง = ส่งกี่/กี่คน (%) + ค้างกี่คน · ข้อสอบ = คะแนนเฉลี่ย/เต็ม · ท้ายสุด = ค้างรวม
   อัปเดตทันทีตอนคลิก/กรอกคะแนน
5. **รายงานรายชิ้นงาน + ส่ง LINE** ⭐ (ฟีเจอร์หลักที่ขอ)
   - กดลิงก์ 📤 ในแถวสรุปท้ายตาราง หรือปุ่ม "📤 รายงาน/LINE" ในแท็บจัดการงาน
   - **งานส่ง** → รายชื่อทุกคนพร้อมสถานะ ✓ ส่งแล้ว / ✗ ยังไม่ส่ง + สถิติ + กล่องสรุปคนยังไม่ส่ง
   - **ข้อสอบ** → รายชื่อพร้อมคะแนนรายคน + สถิติ (เฉลี่ย/สูงสุด/ต่ำสุด/ทำข้อสอบ/ขาดสอบ) + กล่องสรุปคนขาดสอบ
   - มีตัวอย่างรูปในกล่อง → กด 🖼️ ส่งรูปเข้า LINE (มือถือแชร์ / คอมดาวน์โหลด PNG)
6. **แท็บสรุป (📈)** อัตราส่งรวม, อัตราส่งรายชิ้น, ค่าเฉลี่ยข้อสอบ, รายชื่อคนค้างส่ง + ปุ่มส่งรูป LINE
7. **แท็บสรุปรวมทุกห้อง (🗂️)** รวมทุกห้อง/วิชา: สถิติรวม + การ์ดรายห้อง + คนค้างส่งรวมทุกห้อง
8. **📊 Export Excel ทั้งหมด** (อยู่ในแท็บสรุปรวม) — ไฟล์ .xlsx มีชีต "สรุปรวม" + ชีตแยกรายห้อง
   (รายชื่อ × งาน, ส่ง/ไม่ส่ง/คะแนน, คอลัมน์ค้างส่ง, แถวสรุปท้าย)
9. **สำรอง/กู้คืนข้อมูล** เป็นไฟล์ .json (กันข้อมูลหายเวลาเปลี่ยนเครื่อง/ล้างเบราว์เซอร์)
